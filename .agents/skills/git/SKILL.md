# git

## Description

This skill handles semantic commit operations in Git repositories.

## Usage

- Use this skill to create semantic commits following conventional commit guidelines
- Automatically generate commit messages based on code changes
- Stage modified files and push commits to the current branch

## Steps

6. commit message must be small_case
1. Detect modified files in the working directory
2. Generate a semantic commit message based on code changes
3. Stage the modified files
4. Commit the changes with the generated message
5. Push the commit to the current branch
